## 一个基于gulp的前端自动化的基础框架
- 所有的gulp插件配置都写在gulpfile.js中
## how to use ?

<pre>
  <code>
    $ npm install
  </code>
</pre>

## gulp插件

* gulp-uglify `js压缩混淆`
* gulp-concat `合并多个文件`
* gulp-clean-css `压缩css为一行`
* gulp-less `less编译为css`
* gulp-htmlmin `html压缩`
* gulp-changed `检查文件是否更改`
* gulp-smushit `Yahoo开发的jpg/png优化插件`
* gulp-imagemin `图片压缩`
* gulp-debug `debug`
* del `删除文件或者文件夹`
